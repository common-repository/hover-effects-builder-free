Hover Effects Builder is a brand new WordPress plugin for creating modern and elegant CSS hover effects and animations.


== Description ==
Hover Effects Builder includes more than 60 templates so that you can add hover effects to the images on your websites. Hover Effects Builder also features a built-in editor that is available in PRO version. 

<h4>With Hover Effects Builder you are able to:</h4>
<ul>
    <li>Apply CSS3 hover effects to the images on your website without any CSS coding knowledge.</li>
    <li>Assign any of predefined effects to featured images for standard and custom post formats.</li>
    <li>Apply hover effects via fully customizable shortcode to the images in WordPress Post/Page Editor.</li>
</ul>

Please follow the steps below to install Hover Effect Builder in WordPress:
<ol>
    <li>Install the plugin through the WordPress plugins screen directly.</li>
    <li>Activate the plugin through the 'Plugins' screen in WordPress.</li>

</ol>


== Frequently Asked Questions ==

= How to Create/Edit Templates? =
To create new template press "New Template" button and you'll be redirected to Editor Tab to create unique hover effects without coding.

= Can I save edited hover effect templates? =
No, this feature is available in PRO version only.

= Can I apply the predifined templates to my images? =
Yes, you can.

= Can I create new hover effects =
Yes, but you will not be able to save them.
